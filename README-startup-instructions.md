# Workflow for Bennet Site

1. `npm run dev` to dev the actual CMS (Sanity)

Change schemas in `schemas`

2. `npm run build` to build the product
3. `npm run deploy` to deploy
4. `npm run deploy-graphql` to deploy the CMS structure to GraphQL, which the frontend uses
